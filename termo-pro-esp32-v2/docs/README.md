Components used:

**** Thermostat Unit ****

- 1 x WC1602A Display;
- 1 x ESP32WROOM32-NodeMCU32S 38pin Devkit;
- 1 x DHT11 Temperature&Humidity Sensor;
- 1 x Buzzer;
- 1 x 8PIN_P2.54mm USB-C Soldered module;
- 1 x 1N4007 Diode(smaller one work too);
- 4 x 6mm tactile buttons;
- 2 x 10k resistors;
- 1 x 1k resistor;
- 1 x 330 ohm resistor;
- 1 x 10k RM65 potentiometer;
- 1 x 10k LDR Photoresistor;
- 2 x PN2222A BJT NPN Transistors;
- 2 x 50V 220nF capacitors;
- 1 x 16V 680uF capacitor;
- 1 x 25V 220uF capacitor;


**** Starting Unit ****

- 1 x ESP32-C3-Supermini Devkit 16pin;
- 1 x 8PIN_P2.54mm USB-C Soldered module;
- 3 x 220 ohm resistors; *P1*
- 1 x 330 ohm resistor;
- 1 x 10k resistor;
- 1 x 25V 220uF capacitor;
- 1 x 50V 220nF capacitor;
- 1 x 10V 470uF capacitor;
- 1 x 12mm tactile LED(1.8V) button; *P1*
- 1 x PN2222A BJT NPN Transistor;
- 1 x Rectangular GREEN LED(1.8V); *P1*
- 1 x Rectangular RED LED(1.8V); *P1*
- 2 x Screw Terminal Blocks P5.08 - 5mm;
- 1 x 1N4007 Diode;
- 1 x SRD_FORM_C 5V Relay;


*P1* -> Resistor values for each LED is calculated based on the internal diode forward voltage using Ohm Law !
I recommand having LED of the same forward voltage, so you can get equal brightness.
